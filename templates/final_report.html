🎓 University Domain Ontology Project

Final Project Report

Advanced Semantic Web Technologies & Ontology Engineering
Executive Summary

This report details the design and implementation of a comprehensive Knowledge Representation and Reasoning (KRR) system tailored for the university domain. Leveraging Semantic Web technologies, it features an OWL-based ontology, semantic querying via SPARQL, interactive visual analytics, and a responsive web interface. The system embodies the practical application of ontology engineering, offering deep insights into academic structures and operations.

Key Achievements
Ontology Design: 6 departments, 30 students, 19 professors, 35 courses
Visualization: 8 interactive, real-time charts
SPARQL Interface: Powerful semantic query capabilities
User Interface: Fully responsive, professional UI/UX
Semantic Reasoning: RDFLib-powered reasoning and inference

5.2 Example Query
PREFIX university: <http://www.semanticweb.org/university#> SELECT ?name ?email ?gpa ?department WHERE { ?student university:hasName ?name ; university:hasEmail ?email ; university:hasGPA ?gpa ; university:belongsTo ?dept . ?dept university:hasName ?department . } ORDER BY DESC(?gpa)
